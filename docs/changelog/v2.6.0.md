# 版本 v2.6.0 更新日志

## 📅 发布日期
2026-05-22

## 🗑️ 功能移除

### 移除笔记功能
- 删除 NotePage.vue 页面文件
- 删除 Note.js 数据模型
- 删除 note.js 后端路由文件
- 移除 /note 前端路由配置
- 从侧边导航栏移除"我的笔记"菜单项
- 从用户指南移除笔记功能说明章节
- 从登录页移除"智能笔记"功能卡片
- 移除学习数据统计中的笔记相关统计
- 移除 AI答疑功能中的"保存到笔记"相关文案

## 🔧 技术改进

### 后端优化
- 移除 /api/note 路由挂载
- 移除 noteRouter 路由引用
- 优化学习数据统计接口，移除笔记数据查询
- 更新 studyData.js 路由，移除 Note 模型依赖
- 更新后端功能描述，移除"学习笔记"相关说明

### 前端优化
- 简化导航结构，移除笔记功能入口
- 优化学习数据页面，移除笔记统计卡片
- 更新用户指南，移除笔记使用教程
- 更新隐私政策，移除笔记内容相关说明
- 更新服务条款，移除笔记相关知识产权说明
- 精简功能导航，提升用户体验

## 📝 路由变更

### 移除路由
- DELETE /note - 笔记页面路由
- DELETE /api/note/* - 所有笔记相关后端接口

## 📊 数据变更

### 学习数据统计
- 移除 noteCount 字段
- 移除笔记相关的学习明细记录
- 保留学习计划和错题统计功能

## ⚠️ 注意事项
- 用户访问 /note 路由将无法找到页面
- 数据库中的笔记数据不会自动删除，需手动清理
- 建议使用错题集和学习计划功能记录学习内容
- AI助手对话记录功能保持不变
