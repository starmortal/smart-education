# 更新日志 v2.8.3

**版本号**: v2.8.3  
**发布日期**: 2026-05-22

## 🎨 界面优化

### 错题集页面重构
- 重构页面布局：采用类似笔记页面的伸缩侧边栏设计
- 状态筛选按钮移至左侧顶部，支持点击取消筛选
- 添加/管理按钮移至左侧顶部，仅显示图标
- 右侧内容区域支持侧边栏伸缩，提供更大的显示空间
- 优化科目筛选项样式：缩小边框和内边距，使显示更紧凑
- 修复错题卡片中科目名称显示不完整的问题
- 改进整体视觉效果和交互体验

### 学习计划页面重构
- 移除顶部统计栏，采用与错题集一致的布局设计
- 添加伸缩侧边栏，包含状态筛选和科目筛选
- 状态筛选（未开始、进行中、已完成、已逾期）改为卡片样式
- 科目筛选列表支持点击取消筛选
- 右侧内容区域支持侧边栏伸缩
- 顶部采用简洁的横线分隔样式
- 统一全站页面布局风格

### 笔记和意见反馈页面顶部UI统一
- 统一笔记页面顶部样式，使用 `editor-header` 和 `file-name` 类
- 优化意见反馈页面顶部UI：
  - 提交表单页面顶部改为统一的 `editor-header` 样式
  - 详情页面顶部改为统一的 `editor-header` 样式
  - 标签大小调整为 `small`，与其他页面保持一致
  - 时间显示字体大小调整为 13px
- 确保所有页面顶部高度和样式完全统一
