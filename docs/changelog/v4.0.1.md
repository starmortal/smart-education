## 🐛 Bug 修复

### 个人中心 AI 成绩分析

- **修复 404**：补全缺失的后端接口，个人中心「AI 成绩分析」可正常调用
- **接口列表**：
  - `POST /api/ai/analyze-scores-stream` — SSE 流式分析
  - `POST /api/ai/save-score-analysis` — 保存分析记录
  - `GET /api/ai/score-analysis-list` — 历史列表
  - `GET /api/ai/score-analysis/:id` — 分析详情
  - `DELETE /api/ai/score-analysis/:id` — 删除记录
- **AI 模型**：统一使用硅基流动 OpenAI 兼容接口（DeepSeek），默认 `deepseek-ai/DeepSeek-V3`，可通过 `.env` 中 `OPENAI_BASE_URL`、`OPENAI_MODEL` 覆盖
