# 版本 v4.3.3 更新日志

**发布日期**: 2026-05-23

---

## 🎯 本次更新概述

首页页脚「联系我们」改为「发展规划」，新增平台发展规划文档，涵盖产品路线图、积分体系融入方案与商业发展规划，并在平台文档站左侧列表中置于「使用文档」下方。

---

## ✨ 新功能

### 发展规划文档（`/platform-docs?doc=development-roadmap`）

- 文档路径：`docs/guide/development-roadmap.md`
- 内容包含：
  - 平台愿景与定位
  - 产品功能分阶段规划（近期 / 中期 / 长期）
  - **积分体系**获取、消耗、技术路径与风控规则
  - **商业发展规划**：会员分层、B 端模式、收入结构、增长策略、里程碑
  - 技术演进与风险应对

### 平台文档站

- 左侧导航新增「发展规划」，排序位于「使用文档」之后
- 旧路径 `/development-roadmap` 自动重定向至平台文档页

### 首页页脚

- 「支持」栏目：「联系我们」→「发展规划」，点击跳转文档页

---

## 🐛 问题修复

### Socket.IO 连接竞态

- 修复退出登录或重复连接时，`connect` 回调访问已清空的 `this.socket.id` 导致页面报错
- 连接管理改为实例校验 + 断开前移除监听；侧边栏退出时主动断开 Socket
