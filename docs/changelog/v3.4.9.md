# 版本 v3.4.9 更新日志

**发布日期**: 2026-05-23

---

## 🎯 本次更新概述

本版本优化学习社区「我的问题」浏览与管理体验：主区域左滑切换列表与详情，不再使用弹窗查看自己的问题。

---

## 🎯 体验优化

### 1. 我的问题 / 我的收藏内联详情与左滑动画
- **列表展示**: 点击左侧「我的问题」或「我的收藏」后，在中间主区域直接展示列表（带进入动画）
- **左滑详情**: 点击某条问题后，列表向左滑出，详情从右侧滑入，全程无弹窗
- **返回导航**: 详情顶栏提供「返回我的问题」/「返回我的收藏」，滑回列表
- **管理操作**: 我的问题支持标记已解决、删除；收藏支持取消收藏；详情内支持回答、点赞、设最佳答案
- **选中高亮**: 当前查看的问题在列表中高亮显示

### 2. 问题详情组件抽取
- 新增 `CommunityQuestionDetail` 组件，供内联详情与社区弹窗详情复用
