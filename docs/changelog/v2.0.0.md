# v2.0.0 - 界面重构与AI助手功能

**发布日期**: 2026-05-21

## 🎨 界面重构
- 将顶部导航栏改为左侧垂直导航栏
- 左侧导航栏宽度70px，只显示图标，鼠标悬停显示功能名称
- 导航栏固定在页面左侧，主内容区域自适应右侧空间
- 优化移动端适配，导航栏宽度调整为60px
- 所有页面统一使用新的左侧导航布局
- 移除所有页面的TopNavBar组件引用

## 🤖 AI助手功能
- 新增AI助手对话页面（/ai-chat），作为登录后的默认首页
- 实现助手管理功能：创建、编辑、删除助手
- 实现话题管理功能：创建、重命名、删除话题
- 支持多助手、多话题的层级管理
- 每个助手可配置独立的提示词和参数
- 话题自动保存对话历史
- 支持Markdown渲染，优化消息显示
- 集成DeepSeek AI，实现智能对话
- 自动创建默认助手，简化首次使用流程
- 采用标签切换式布局：左侧面板支持「助手」和「对话」标签切换
- 点击助手自动切换到对话标签，显示该助手的话题列表
- 动态按钮：根据当前标签显示「新建助手」或「新建对话」
- 优化交互流程，简化操作步骤

## 🔧 后端新增
- 新增Assistant数据模型（助手管理）
- 新增Topic数据模型（话题管理）
- 新增assistantController控制器
- 新增chatController控制器

- 完善错误处理和日志记录

## 📱 前端新增
- 新增SideNavBar组件（左侧导航栏）
- 新增AIChat页面（AI助手对话主页面）
- 新增assistant.js API接口层
- 新增chat.js API接口层
- 修改App.vue为左右布局结构
- 更新路由配置，默认跳转到/ai-chat

## 🎯 用户体验优化
- 左侧导航栏提供快速功能切换
- AI助手页面采用两栏布局：左侧标签面板（260px）、右侧对话区域
- 左侧面板支持「助手」和「对话」标签切换，简化界面层级
- 点击助手自动切换到对话标签，交互更流畅
- 对话标签显示当前助手信息，便于识别上下文
- 支持实时消息发送和接收
- 话题标题自动生成，可手动重命名
- 消息时间显示优化
- 加载状态提示优化

## 📊 技术改进
