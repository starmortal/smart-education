# 版本 v2.7.0 更新日志

## 📅 发布日期
2026-05-22

## ✨ 新增功能

### 笔记功能重新实现
- 全新设计的笔记页面，采用左右分栏布局
- 左侧笔记列表区，右侧Markdown编辑区
- 使用 md-editor-v3 作为Markdown编辑器

### 核心功能
- **笔记管理**：创建、编辑、删除笔记
- **Markdown编辑**：支持富文本工具栏，实时预览
- **分类管理**：支持自定义分类，快速筛选
- **标签系统**：支持多标签，可创建和筛选
- **全文搜索**：支持标题和内容的全文搜索
- **自动保存**：编辑后1秒自动保存，防止数据丢失

### 编辑器功能
- 支持加粗、斜体、下划线、删除线
- 支持标题、引用、列表（有序/无序）
- 支持代码块、行内代码、链接、图片
- 支持表格插入
- 支持全屏编辑和预览模式
- 支持目录导航

## 🔧 技术实现

### 前端部分
- **新增文件**：`frontend/src/views/NotePage.vue`
- **新增依赖**：md-editor-v3（Vue 3 Markdown编辑器）
- **路由配置**：添加 /note 路由
- **API接口**：完整的笔记CRUD接口
- **导航菜单**：在侧边栏和顶部导航添加笔记入口

### 后端部分
- **数据模型**：`smart-education-backend/models/Note.js`
  - userId: 用户ID
  - noteTitle: 笔记标题
  - noteContent: 笔记内容（Markdown格式）
  - noteCategory: 分类
  - noteTags: 标签数组
  - createTime: 创建时间
  - updateTime: 更新时间

- **路由接口**：`smart-education-backend/routes/note.js`
  - GET /api/note/list - 获取笔记列表（支持搜索、筛选）
  - POST /api/note/add - 新建笔记
  - PUT /api/note/update/:id - 更新笔记
  - DELETE /api/note/delete/:id - 删除笔记
  - GET /api/note/categories - 获取分类列表
  - GET /api/note/tags - 获取标签列表

## 💡 使用说明

### 创建笔记
1. 点击"新建笔记"按钮
2. 在右侧编辑区输入标题和内容
3. 选择分类和标签
4. 系统自动保存

### 搜索笔记
1. 在搜索框输入关键词
2. 支持搜索标题和内容
3. 实时过滤显示结果

### 分类管理
1. 在编辑区选择分类下拉框
2. 选择"+ 新建分类"创建新分类
3. 输入分类名称确认

### 标签管理
1. 在标签选择器中选择已有标签
2. 或直接输入新标签名称创建
3. 支持多标签选择

## ⚠️ 注意事项
- 笔记内容使用Markdown格式存储
- 自动保存间隔为1秒（编辑停止后）
- 切换笔记前会自动保存当前笔记
- 删除操作需要确认，删除后无法恢复
- 搜索功能使用防抖处理，输入停止0.5秒后执行
