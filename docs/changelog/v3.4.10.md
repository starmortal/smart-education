# 版本 v3.4.10 更新日志

**发布日期**: 2026-05-23

---

## 🎯 本次更新概述

本版本为学习社区补齐「全部问题」等列表的左滑详情体验，并上线嵌套回复与更紧凑的评论区样式。

---

## ✨ 功能优化

### 1. 回答嵌套回复（评论区样式）
- **嵌套回复**: 支持对回答及楼中楼进行回复，展示「回复 @用户名」
- **两级结构**: 一级评论与次级回复分层展示，风格参考短视频评论区
- **交互**: 每条回答可点「回复」展开行内输入框；底部输入框发表一级评论
- **后端**: `Answer` 模型新增 `parentId`、`replyToUserId`、`replyToUserName` 字段

### 2. 全部问题左滑查看详情
- **统一体验**: 「全部问题」「关注的人」「为你推荐」点击问题后，与「我的问题 / 我的收藏」一致，主区域左滑进入详情，不再使用弹窗
- **返回导航**: 按当前筛选显示「返回全部问题」等文案
- **列表高亮**: 正在查看的问题在列表中高亮
- **移除详情弹窗**: 学习社区内问题详情均改为内联左滑展示

### 3. 评论区视觉精简
- **一级回复**: 缩小头像、字号与内边距，整体更紧凑
- **次级回复**: 进一步缩小并缩进，层级更清晰
- **二次优化**: 一级/嵌套回复字号、头像与操作按钮再度微调，更接近短视频评论区密度

### 4. 组件
- 新增 `CommunityAnswerThread` 递归评论组件
- 新增 `CommunitySlideDetailPanel` 滑动详情容器
- 新增 `answerTree` 工具函数构建评论树
