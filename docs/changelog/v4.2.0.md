# 版本 v4.2.0 更新日志

**发布日期**: 2026-05-23

---

## 🎯 本次更新概述

重构管理后台为独立控制台：新账号登录、数据可视化概览、全站系统公告推送，与用户端侧边栏完全分离。

---

## ✨ 新功能

### 独立管理后台布局

- 深色侧边栏 + 顶栏，不再显示用户端 `SideNavBar`
- 菜单：数据概览、反馈管理、系统公告
- 支持从管理端返回用户端

### 管理员登录

- 固定账号：`admin` / `admin123456`
- 对接后端 `POST /api/admin/login` 校验

### 数据统计与图表

- 概览卡片：全平台用户数、社区提问数、反馈数、问题解决情况
- **折线图**：近 7/30/90 天用户注册与提问增长趋势
- **饼图**：提问解决状态、反馈类型分布、用户年级分布

### 全站系统公告

- 管理员可编辑标题与内容，一键推送给全平台用户
- 用户可在「通知中心」查看（类型 `system`，尊重用户通知开关）

### 反馈管理（保留并增强）

- 列表筛选、查看详情、回复功能保留
- API 迁移至 `/api/admin/feedback/*`，需管理员 Token 鉴权

---

## 🔧 后端改动

- 新增 `routes/admin.js`、`services/adminService.js`、`middleware/adminAuth.js`

## 🎨 界面优化

- 管理后台整体紧凑化：侧边栏 168px、顶栏/卡片/字体/图表高度同步缩小
- 修复管理后台切换页面后图表报错、表单校验未捕获导致的运行时错误

### 通知中心

- 点击系统公告改为弹窗查看详情，不再跳转到个人中心
- 列表摘要自动去除 HTML 标签，显示更整洁
