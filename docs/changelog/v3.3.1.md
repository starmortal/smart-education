## 📝 技术说明

### 问题原因
与错题本模块相同，后端使用统一响应格式返回数据：
```json
{
  "code": 200,
  "message": "操作成功",
  "data": { ... },
  "timestamp": 1234567890
}
```

前端 axios 响应结构为 `res.data`，因此实际数据在 `res.data.data` 中。

### 修复方案
在所有学习计划相关的数据加载函数中，统一使用以下方式获取数据：
```javascript
const listData = res.data?.data || res.data;
```

这样既兼容新的统一响应格式，也兼容旧的直接返回数据格式。
