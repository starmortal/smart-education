# 版本 v3.4.5 更新日志

**发布日期**: 2026-05-23

---

## 🎯 本次更新概述

本版本参照**学习社区问题卡片**的极简社交流式布局，重构错题集页面右侧错题展示卡片，统一平台视觉风格，提升阅读与操作体验。

---

## 🎨 UI优化

### 1. 错题卡片布局重构
- **参照对象**: 学习社区 `question-card` 极简卡片样式
- **布局调整**: 右侧列表改为居中 feed 流（最大宽度 720px），背景色与学习社区一致（`#f9fafb`）
- **卡片结构**:
  - 顶部元信息行：科目圆形图标 + 科目名 + 题型/添加时间
  - 错题标题（18px 加粗）
  - 错误原因预览（最多 100 字，超出省略）
  - 科目标签 + 题型标签（pill 样式）
  - 渐变分隔线
  - 底部统计栏：查看详情、掌握状态徽章、操作按钮（标记掌握/编辑/删除）
- **交互效果**: 悬停上浮阴影、圆角 16px、左侧 3px 科目色边框；已掌握错题左边框变为绿色

### 2. 空状态与分页优化
- **空状态**: 采用与学习社区一致的空状态样式（图标 + 主文案 + 提示文案）
- **分页**: 移至列表底部居中，使用 `background` 样式分页器，与学习社区分页风格统一

### 3. 样式修复
- **修复**: 将原先写在 `</style>` 标签外的错题卡片样式移入 `scoped` 作用域，避免样式泄漏

### 4. 错题列表双列布局
- **布局调整**: 右侧错题卡片由单列 feed 改为双列网格（`grid-template-columns: repeat(2, 1fr)`）
- **滚动体验**: 列表区域保持纵向滚动，内容可连续向下浏览
- **响应式**: 窄屏（≤768px）自动回退为单列显示

### 5. AI 错题分析 Markdown 样式统一
- **样式对齐**: AI 分析弹窗 Markdown 渲染样式与个人中心「AI 成绩分析」保持一致
- **标题美化**: h1/h2/h4 使用主题蓝色（`#0969da`），h2 带底部分割线
- **强调样式**: `strong` 关键字高亮为蓝色
- **引用块**: 左侧蓝色竖线 + 浅灰背景
- **流式输出**: 分析过程中底部显示蓝色加载提示条，支持边输出边预览
