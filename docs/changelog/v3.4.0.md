# 版本 v3.4.0 更新日志

**发布日期**: 2026-05-23（开发中）

## 🎯 版本概述

本版本对**学习计划页面**和**错题集页面**进行重大升级，引入AI智能辅助功能和全新的视图模式，旨在提升学习效率和用户体验。

### 核心升级
- **学习计划页面**：采用时间轴视图 + AI学习助手
- **错题集页面**：融合复习日历 + 知识图谱视图
- **AI深度集成**：智能建议、遗忘风险评估、知识点分析

## 📚 学习计划页面升级

### 新增功能

#### 1. AI学习助手面板
- **智能问候**：根据时间和学习状态个性化问候
- **今日任务概览**：显示今天需要完成的学习任务
- **AI建议**：基于学习进度和历史数据提供智能建议
- **快捷操作**：一键开始学习、查看详情、调整计划
- **可折叠设计**：支持最小化，节省屏幕空间

#### 2. 时间轴视图
- **时间分组**：按今天、明天、本周、本月自动分组
- **可视化展示**：清晰的时间线布局，一目了然
- **折叠/展开**：每个时间段支持折叠，灵活查看
- **智能排序**：按紧急程度和时间自动排序

#### 3. 增强版计划卡片
- **紧急程度标识**：🔥紧急、⏰进行中、📅计划
- **彩色进度条**：动态显示学习进度
- **AI建议区域**：
  - 今日目标和预计时长
  - 学习资源推荐
  - 最佳学习时段
  - 个性化提示
- **快捷操作按钮**：开始学习、继续学习、暂停、查看详情

#### 4. 周统计概览
- **本周学习统计**：总计划数、进行中、已完成、平均进度
- **AI周报**：与其他用户对比，给予鼓励和建议
- **趋势分析**：显示学习进度变化趋势

### 优化改进
- **左侧筛选栏**：添加"AI推荐"筛选项
- **状态计数**：实时显示每个状态的计划数量
- **多选筛选**：支持同时选择多个筛选条件

## 📝 错题集页面升级

### 新增功能

#### 1. 复习日历
- **日历视图**：显示一周/一月的复习情况
- **每日统计**：显示每天需复习的错题数量
- **完成标记**：已完成的显示✓标记
- **今日高亮**：当前日期特殊标识
- **点击交互**：点击日期查看当天的错题

#### 2. AI复习建议面板
- **今日任务**：显示今日需复习的错题数量和预计时长
- **重点突破**：AI分析需要重点复习的知识点
- **复习策略**：提供科学的复习方法建议
- **快捷操作**：开始复习、查看知识图谱、生成复习计划

#### 3. 知识图谱视图（核心功能）
- **可视化展示**：用节点和连线展示知识点关系
- **节点大小**：表示该知识点的错题数量
- **颜色编码**：红色（薄弱）→黄色（一般）→绿色（掌握）
- **交互功能**：
  - 点击节点查看相关错题
  - 支持缩放和拖拽
  - 悬浮显示详细信息
- **AI诊断**：
  - 识别核心问题和根本原因
  - 规划学习路径
  - 预测突破时间

#### 4. 增强版错题卡片
- **优先级标识**：🔴紧急、🟡重要、🟢巩固
- **遗忘风险评估**：基于艾宾浩斯遗忘曲线计算
- **掌握度显示**：综合复习次数和正确率评估
- **复习统计**：显示添加时间、复习次数
- **AI个性化提示**：针对每道题的具体建议
- **快捷操作**：开始复习、查看解析、延后、已掌握

#### 5. 视图切换
- **双视图模式**：知识图谱视图 ↔ 列表视图
- **一键切换**：右上角切换按钮
- **偏好记忆**：保存用户的视图偏好

### 智能算法

#### 遗忘风险计算
基于艾宾浩斯遗忘曲线：
- 复习间隔：1天、2天、4天、7天、15天
- 动态调整：根据复习次数自动调整间隔
- 风险评分：0-100%，超过60%建议立即复习

#### 掌握度评估
综合考虑：
- 复习次数（基础分：每次20分，最高60分）
- 正确率（加分：正确率×40分）
- 最终得分：0-100%

### 前端新增组件

- `TimelineView.vue` - 时间轴视图

### 后端新增接口

**学习计划相关**：
- `GET /api/study-plan/ai-suggestion/:planId` - 获取AI学习建议
- `GET /api/study-plan/timeline/:userId` - 获取时间轴数据

**错题集相关**：
- `GET /api/error-book/calendar/:userId` - 获取复习日历数据

- `GET /api/error-book/ai-review-plan/:userId` - 获取AI复习建议
- `POST /api/error-book/calculate-risk` - 计算遗忘风险

### 使用的技术和库
- **ECharts 6.0.0**：知识图谱可视化
- **Day.js**：日期处理和计算
- **OpenAI API**：AI建议生成
- **Element Plus**：UI组件库
