# v2.2.0 - 文件上传功能

**发布日期**: 2026-05-21

## 📤 文件上传功能
- 完整的文件上传功能实现
- 支持图片格式：JPG、JPEG、PNG、GIF、WebP
- 支持文档格式：PDF、Word（DOC/DOCX）、TXT
- 文件大小限制：10MB
- 支持多文件同时上传

## 🎨 文件预览
- 上传前文件预览功能
- 图片显示缩略图（100x100px）
- 文档显示文件图标和文件信息
- 可删除已选择的文件
- 文件大小自动格式化显示

## 💬 消息附件显示
- 消息中显示附件信息
- 图片附件可点击放大查看
- 文档附件提供下载链接
- 附件与消息内容分离显示
- 支持多个附件同时展示

## 🔧 后端实现
- 新增文件上传接口：`POST /api/chat/upload`
- 使用 Multer 处理文件上传
- 文件存储在 `uploads/chat/` 目录
- 文件名使用时间戳+随机数避免冲突
- 文件类型和大小验证
- 静态文件服务配置

## 🔒 安全措施
- 文件类型白名单验证
- 文件大小限制（10MB）
- 文件名安全处理
- MIME 类型检查

## 📱 用户体验
- 文件选择后即时预览
- 上传进度提示
- 错误提示友好
- 支持拖拽上传（文件选择框）
- 可同时上传多个文件

## 🔗 影响范围
- 修改文件：
  - 前端：
    * `frontend/src/views/AIChat.vue`（文件上传UI和逻辑）
    * `frontend/src/api/chat.js`（新增上传接口）
  - 后端：

    * `smart-education-backend/controllers/chatController.js`（上传和消息处理）

  - `smart-education-backend/uploads/chat/`（文件存储目录）
- 新增文档：
  - `docs/changelog/v2.2.0.md`

## 🐛 Bug 修复
- 修复图片上传预览显示为 "preview" 文本的问题
  - 将 FileReader 异步读取改为 Promise 方式，确保图片完全加载后再显示
  - 优化图片 alt 属性，使用文件名而非固定文本
  - 新增 `readFileAsDataURL` 辅助函数处理异步文件读取
  - 修复时间：2026-05-21

## 🎨 UI 细节优化
- 左侧导航栏整体缩小优化：
  - 侧边栏宽度从 70px 缩小到 60px
  - Logo 区域高度从 70px 缩小到 60px
  - Logo 图标从 40px 缩小到 28px
  - 导航按钮高度从 60px 缩小到 50px
  - 导航图标从 24px 缩小到 18px
  - 用户头像从 40px 缩小到 32px
  - 整体视觉更精致紧凑，节省屏幕空间

## ⚠️ 注意事项
- 文件存储在服务器本地 `uploads/chat/` 目录
- 需要确保该目录有写入权限
- 文件通过 `/uploads/chat/:filename` 访问
- 消息数据结构新增 `attachments` 字段
- 旧版本消息兼容（attachments 为可选字段）

## 🚀 使用说明
1. 点击输入框下方的回形针图标选择文件
2. 支持选择多个文件
3. 选择后会显示文件预览
4. 可点击预览卡片上的 X 删除文件
5. 点击发送按钮上传文件并发送消息
6. 图片可在消息中点击查看大图
7. 文档可点击下载
