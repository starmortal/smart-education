# v2.11.0 更新日志

**发布日期**: 2026-05-22

## ✨ 新功能

### 学习社区模块 - 极简社交流式设计
- **右侧内容区域全新改版**
  - 采用极简主义设计理念，结合社交媒体交互模式
  - 从 2x2 网格布局改为单列流式布局
  - 提升信息密度和阅读体验

## 🎨 界面优化

### 问题卡片重设计
- **极简视觉风格**
  - 去除彩色边框，采用柔和阴影和大圆角设计
  - 卡片圆角：16px，阴影：0 2px 8px rgba(0,0,0,0.04)
  - 悬停效果：轻微上移 + 阴影加深
  - 流畅的过渡动画（cubic-bezier 缓动函数）

- **优化的信息层级**
  - 标题：18px / 600 字重
  - 内容预览：15px / 400 字重
  - 元数据：13px / 400 字重
  - 标签：12px / 500 字重

- **现代化配色方案**
  - 主文本：#1a1a1a（深灰替代纯黑）
  - 次要文本：#6b7280（中灰）
  - 强调色：#3b82f6（柔和蓝）
  - 成功色：#10b981（柔和绿）
  - 警告色：#f59e0b（柔和橙）

### 新增内容展示
- **问题内容预览**
  - 显示问题描述前 100 字符
  - 超出部分显示省略号
  - 提升用户对问题的理解

- **丰富的互动统计**
  - 回答数统计（💬 图标）
  - 浏览数统计（👁 图标）
  - 状态徽章（✓ 已解决 / ⏱ 待解决）
  - 图标 + 文字组合，清晰直观

- **热门回答预览区域**（预留功能）
  - 渐变背景设计
  - 显示最热门回答的前 80 字符
  - 包含回答者和点赞数
  - 点击可直接查看完整回答

### 标签系统优化
- **胶囊式标签设计**
  - 半透明背景：rgba(59, 130, 246, 0.08)
  - 圆角：12px
  - 悬停效果：背景加深
  - 最多显示 3 个标签，超出显示 "+N"

### 用户信息展示优化
- **更大的头像**：从 32px 增加到 40px
- **双行元数据**：
  - 第一行：用户名（15px / 600 字重）
  - 第二行：年级 · 发布时间（13px / 灰色）
- **收藏图标优化**：
  - 未收藏：#d1d5db（浅灰）
  - 已收藏：#3b82f6（蓝色填充）
  - 悬停放大效果（scale: 1.15）

### 空状态设计
- **优雅的空状态页面**
  - 大尺寸图标（80px）
  - 主提示文字：18px / 600 字重
  - 副提示文字：14px / 灰色
  - 引导用户调整筛选或发起新问题

### 分页组件优化
- **现代化分页样式**
  - 圆角按钮：8px
  - 背景模式启用
  - 活动页码：蓝色背景 + 白色文字
  - 居中对齐，上下留白

## 🔧 布局优化

### 单列流式布局
- **最大宽度限制**：720px，内容居中显示
- **卡片间距**：20px
- **内边距**：24px（卡片内部）
- **外边距**：24px 20px（容器）

### 响应式适配
- **桌面端（>1200px）**：
  - 最大宽度 720px，居中显示
  - 完整的卡片内边距

- **平板端（768px-1200px）**：
  - 最大宽度 100%
  - 调整外边距为 20px 16px

- **移动端（<768px）**：
  - 减少卡片内边距至 20px
  - 减少卡片间距至 16px
  - 优化字号和间距

## 📊 数据展示

### 每页显示数量调整
- **修改前**：4 条（2x2 网格）
- **修改后**：6 条（单列流式）
- **原因**：单列布局信息密度更高，适合展示更多内容

## 🎯 用户体验提升

### 交互优化
- **流畅的动画效果**
  - 卡片悬停：上移 2px + 阴影加深
  - 图标悬停：放大 1.15 倍
  - 标签悬停：背景加深
  - 过渡时间：0.3s（卡片）/ 0.2s（小元素）

- **视觉反馈增强**
  - 所有可点击元素都有悬停效果
  - 收藏状态有明确的颜色区分
  - 状态徽章使用语义化颜色

### 阅读体验优化
- **大量留白设计**
  - 减少视觉干扰
  - 提升内容聚焦度
  - 符合现代审美

- **清晰的信息层级**
  - 用户信息 → 标题 → 内容 → 标签 → 统计
  - 分隔线区分不同区域
  - 视觉引导自然流畅
