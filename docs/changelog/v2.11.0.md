# v2.11.0 更新日志

**发布日期**: 2026-05-22

## ✨ 新功能

### 🤝 社交关系系统（核心功能）
- **关注/粉丝功能**
  - 用户可以关注其他用户
  - 查看关注列表和粉丝列表
  - 支持互相关注状态识别
  - 关注按钮三种状态：未关注、已关注、互相关注
  - 实时更新关注状态

- **左侧筛选栏增强**
  - 新增"全部问题"筛选项
  - 新增"关注的人"筛选项（查看关注用户的问题）
  - 新增"我的问题"筛选项
  - 新增社交数据展示：
    - 我的关注数量
    - 我的粉丝数量
    - @我的消息数量（预留）

- **用户主页功能**
  - 点击用户头像/名字打开用户主页
  - 显示用户基本信息（头像、昵称、学校、年级）
  - 显示用户统计数据（提问数、关注数、粉丝数）
  - 显示最近提问列表
  - 支持在主页直接关注/取消关注
  - 显示与当前用户的关注关系

- **问题卡片社交元素**
  - 每个问题卡片显示关注按钮
  - 已关注用户的问题有特殊标识（蓝色左边框）
  - 点击用户信息可查看用户主页

### 🤖 AI 智能助手系统（深度分析）

#### 右侧 AI 助手面板（可折叠）
- **智能推荐模块**
  - 基于用户年级和科目推荐相关问题
  - 显示推荐理由
  - 最多显示 3 条推荐
  - 点击直接跳转到问题详情

- **今日学习建议**
  - 分析用户最近提问情况
  - 统计学科分布
  - 生成个性化学习建议
  - 显示重点关注学科

- **热门话题**
  - 统计最近 7 天的热门标签
  - 显示话题热度（问题数量）
  - 点击话题筛选相关问题
  - 最多显示 8 个热门话题

- **社区动态**
  - 今日新增问题数
  - 问题解决率统计
  - 活跃用户数量
  - 实时更新数据

- **面板交互**
  - 支持折叠/展开
  - 折叠按钮固定在左侧
  - 折叠后不占用空间
  - 流畅的动画效果

#### AI 问题分析（深度功能）
- **自动分析问题**
  - 难度评级（1-5 星）
  - 关键词提取
  - 知识点识别
  - 学习建议生成
  - 快速解答提示

- **相似问题推荐**
  - 基于标签匹配相似问题
  - 最多推荐 5 个相关问题
  - 帮助用户找到已有答案

- **AI 标签推荐**
  - 发布问题时自动分析内容
  - 智能推荐相关标签
  - 支持学科、难度、类型标签
  - 提高标签准确性

#### AI 集成 OpenAI
- 使用 GPT-3.5-turbo 模型
- 专业的教育 AI 助手角色
- 结构化的分析结果
- API 调用失败时使用默认分析
- 分析结果缓存，避免重复调用

### 学习社区模块 - 极简社交流式设计
- **右侧内容区域全新改版**
  - 采用极简主义设计理念，结合社交媒体交互模式

   ```javascript
     followerId: String,      // 关注者ID
   ```

   ```javascript
   {
     questionId: String,      // 问题ID（唯一）
     difficulty: Number,      // 难度 1-5
     keywords: [String],      // 关键词
     knowledgePoints: [String], // 知识点

     suggestion: String,      // 学习建议
     quickAnswer: String,     // 快速解答
     learningPath: [Object]   // 学习路径
   }
   ```
