# 版本 v4.1.0 更新日志

**发布日期**: 2026-05-23

---

## 🎯 本次更新概述

优化「我的计划」时间轴体验：记住分组展开/收起状态、详情弹窗支持删除，并升级展开区域的暗色方正视觉风格。

---

## ✨ 功能新增

### 计划详情删除

- 点击时间轴计划卡片打开「计划详情」后，可直接**删除计划**或**编辑计划**
- 删除前二次确认，成功后自动关闭弹窗并刷新列表与统计

---

## 🎨 UI 优化

### 时间轴展开样式

- 分组标题栏采用方正卡片布局，浅绿渐变背景（`#f6fbf8` → `#eef6f1`），细边框
- 展开内容区为嵌套浅绿面板（`#f7fbf9`），计划卡片白色底 + 绿色左边线强调
- 时间轴区域背景为浅绿渐变（`#f4faf7` → `#eef5f1`），整体清爽明亮

---

## 🎯 体验优化

### 时间线展开状态记忆

- 各日期分组（今天 / 明天 / 未来）的展开/收起状态写入 `localStorage`
- 用户下次进入「我的计划」页面时自动恢复上次浏览状态

---

## 🔧 技术改动

- `frontend/src/components/plan/TimelineView.vue`：localStorage 持久化 + 暗色时间轴样式
- `frontend/src/views/StudyPlan.vue`：详情弹窗删除/编辑入口、`handleDeletePlan` 支持成功回调
