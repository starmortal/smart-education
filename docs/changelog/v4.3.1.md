# 版本 v4.3.1 更新日志

**发布日期**: 2026-05-23

---

## 🎯 本次更新概述

更新日志页左侧版本卡片改为浅蓝色样式；隐私政策、服务条款、使用文档合并为统一文档站页面，从 `docs` 分类目录动态加载 Markdown。

---

## ✨ 新功能

### 平台文档站（`/platform-docs`）

- 左侧切换：隐私政策、服务条款、使用文档
- 中间 Markdown 正文 + 右侧本页目录（与更新日志页相同交互）
- 原路由 `/privacy-policy`、`/terms-of-service`、`/user-guide` 自动重定向至新页面

### 文档目录结构

| 分类 | 路径 |
|------|------|
| 法律条款 | `docs/legal/privacy-policy.md`、`docs/legal/terms-of-service.md` |
| 使用指南 | `docs/guide/user-guide.md` |
| 更新日志 | `docs/changelog/*.md`（保持不变） |

### 后端 API

- `GET /api/platform-docs/list`：文档列表
- `GET /api/platform-docs/:slug`：文档 Markdown 内容

---

## 🎨 界面优化

- 更新日志页左侧版本卡片：浅蓝渐变背景与蓝色强调边框
- 平台文档页复用同一套文档站布局与样式
- 扩充 `docs/guide/user-guide.md`：覆盖注册登录、AI 助手、知识库、错题集、学习计划、笔记、社区、个人中心、通知中心、意见反馈等完整教程
