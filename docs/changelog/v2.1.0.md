# v2.1.0 - AI 聊天页面 UI 优化

**发布日期**: 2026-05-21

## 🎨 界面优化
- 标签按钮改为自定义样式，助手和对话各占一半宽度
- 移除列表项的时间和统计信息显示，界面更简洁
- 优化三个点菜单图标，增大尺寸并增强 hover 效果
- 助手菜单项统一命名：编辑助手、删除助手
- 对话菜单项保持：重新命名、清空消息、删除对话
- 助手名称显示在右侧对话框顶部，而非对话列表顶部
- 参照 Cherry Studio 样式优化布局结构

## 🔧 功能增强
- 简化编辑助手对话框，只保留助手名称和系统提示词两个字段
- 对话框宽度调整为 500px，系统提示词输入框增加到 8 行
- 点击助手时自动创建默认对话并打开聊天页面
- 优化用户体验，减少操作步骤

## 💬 消息显示优化
- 所有消息（用户和 AI）统一在左侧显示
- 用户消息使用浅蓝色背景（#e8f4ff）
- AI 消息使用浅灰色背景（#f5f7fa）
- 移除消息时间戳显示，界面更清爽
- 页面加载时自动创建并打开默认对话

## 🎯 输入框美化
- 输入区域增加白色背景和边框
- 添加圆角和阴影效果
- 输入框和按钮区域增加分隔线
- 整体样式更现代化，与消息区域区分明显
- 快捷键优化：Enter 发送消息，Shift+Enter 换行
- 新增文件上传按钮（支持图片、PDF、Word、文本文件）
- 文件大小限制 10MB

## 📱 交互优化
- 标签按钮支持 hover 和激活状态
- 三个点菜单图标增大到 20px，更易点击
- 菜单图标 hover 时有缩放动画效果
- 点击助手自动切换到对话标签并创建默认对话
- 新助手自动打开聊天页面，无需手动创建对话
- 快捷键优化：Enter 快速发送，Shift+Enter 换行
- 新增文件上传功能，支持图片和文档

## 🔗 影响范围
- 修改文件：
  - 前端：`frontend/src/views/AIChat.vue`
  - 文档：`docs/changelog/v2.1.0.md`（新增）

## ⚠️ 注意事项
- 本次更新为 UI 优化，不涉及后端接口变更
- 编辑助手对话框简化，移除模型、温度等高级参数配置
- 点击助手时会自动创建默认对话（如果没有对话）
- 旧版本创建的助手会使用默认参数值
- 消息显示方式改变，所有消息统一左对齐
