# 版本 v2.9.0 更新日志

## 📅 更新日期
2026-05-22

## 🐛 Bug 修复

### 核心问题修复
- **修复响应数据访问错误**：发现并修复了前端访问后端响应数据的关键问题
  - 问题原因：request.js 的响应拦截器返回 `response.data`，但组件中错误地使用了 `res.data.code` 和 `res.data.data`
  - 正确用法：应该使用 `res.code` 和 `res.data`
  - 影响范围：所有助手和知识库的加载、创建、更新操作

### 助手管理
- 修复助手模型中 knowledgeBases 字段重复定义的问题

- 修复加载助手列表的数据访问路径
- 修复加载话题列表的数据访问路径
- 修复加载消息的数据访问路径
- 修复创建话题的数据访问路径
- 修复发送消息的数据访问路径
- 修复文件上传的数据访问路径

### 知识库管理
- 修复新增知识库后列表不显示的问题（响应数据访问错误）
- 修复加载知识库列表的数据访问路径
