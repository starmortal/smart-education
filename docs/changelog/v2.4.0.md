# 版本 v2.4.0 更新日志

## 📅 发布日期
2026-05-21

## ✨ 新增功能

### 助手知识库关联功能
- 助手可以关联多个知识库作为默认配置
- 在编辑助手对话框中添加"关联知识库"选项
- 支持多选知识库，显示知识库文件数量
- 使用助手时自动检索关联的知识库内容

### 对话级别知识库选择
- 在输入框工具栏添加知识库选择按钮
- 点击按钮弹出知识库选择器
- 支持为单次对话临时选择知识库
- 按钮显示已选择知识库数量徽章
- 对话级别知识库优先级高于助手级别
- 弹窗中显示助手默认知识库数量提示

### 智能知识库检索
- AI回答前自动检索相关知识库内容
- 支持多知识库联合检索
- 检索结果作为上下文注入到AI提示词
- 提高AI回答的准确性和相关性

## 🔧 技术改进

### 后端优化
- Assistant模型新增knowledgeBases字段
- 更新助手控制器支持知识库关联
- 聊天控制器集成知识库检索逻辑
- 实现两级知识库优先级机制

### 前端优化
- 使用Element Plus多选组件实现知识库选择
- 使用Popover组件实现知识库选择弹窗
- 使用Badge组件显示已选知识库数量
- 优化助手编辑对话框UI
- 改进用户交互体验，采用按钮+弹窗模式

## 📝 接口变更

### 助手接口
- PUT /api/assistant/:id 新增knowledgeBases参数

### 聊天接口
- POST /api/chat/message/send 新增temporaryKnowledgeBases参数

## 🐛 问题修复
- 无

## 📚 文档更新
- 更新助手功能说明
- 添加知识库关联使用指南

## ⚠️ 注意事项
- 现有助手数据完全兼容，knowledgeBases字段默认为空数组
- 知识库检索失败不影响正常对话
- 建议为常用助手配置相关知识库以提升回答质量
