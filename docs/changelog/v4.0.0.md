# 版本 v4.0.0 更新日志

**发布日期**: 2026-05-23

---

## 🎯 本次更新概述

本版本为**主版本升级**，在学习计划模块引入完整的「AI 智能制定计划」能力：聚合平台多源学习数据、支持手动/流式生成与定时自动生成，完成后通过通知中心与邮件同步提醒用户。

详细技术方案见：`docs/design/ai-smart-plan.md`

---

## ✨ 功能新增

### 1. AI 智能制定学习计划

- **入口**：学习计划页右侧顶部栏「AI 智能制定」
- **手动模式**：预览 AI 草案 → 勾选/编辑 → 确认导入时间轴
- **数据聚合**：个人资料、考试成绩、错题本、现有计划、AI 对话、笔记、学习社区
- **API**：
  - `GET /api/study-plan/learning-profile`
  - `POST /api/study-plan/ai-generate/preview`
  - `POST /api/study-plan/ai-generate/preview-stream`（SSE 流式）
  - `POST /api/study-plan/ai-generate/confirm`
  - `POST /api/study-plan/ai-generate/adjust`（进度智能调整）

### 2. 定时自动生成计划（核心）

- 用户配置**生成时间**、**每天/每周**、**每次生成条数**
- 后端调度器每分钟检查（北京时间），到点自动：
  1. 聚合学习画像并 AI 生成计划
  2. **自动写入**学习计划时间轴（无需手动确认）
  3. 发送**通知中心**系统通知（Socket 实时推送）
  4. 同步发送**邮件**（需在通知设置中开启系统通知与邮件）
- **API**：`POST /api/study-plan/ai-schedule/run-now`（立即执行一次）
- **服务**：`aiPlanSchedulerService`（`index.js` 启动后自动运行）

### 3. 流式生成与制定依据展示

- 抽屉内实时展示 AI 生成过程
- 计划草案显示 **reason（制定依据）** 与 **dataSources（数据来源标签）**
- 学习画像展示 **薄弱知识点** 聚合

### 4. 错题知识点字段与 AI 自动标注

- `ErrorQuestion.knowledgePoints[]`
- 添加错题时自动 AI 提取知识点（失败则规则兜底）
- `POST /api/error-book/extract-knowledge-points`
- `POST /api/error-book/batch-annotate-knowledge`
- 错题卡片展示知识点标签

### 5. 用户偏好持久化

- `User.aiPlanSettings`：开关、数据来源、周期、重点科目、自动调整、**定时规则**
- `GET/PUT /api/study-plan/ai-settings`
- 抽屉配置变更自动保存

### 6. 计划进度智能调整

- 分析逾期与进度落后计划，生成补强建议
- 支持预览与应用；开启后打开抽屉可自动分析

### 7. 学习画像聚合层

- `learningProfileService` 统一跨模块数据汇总
- `utils/subjectMap.js` 科目中英文映射

## ⚠️ 使用说明

1. 在「AI 智能制定」中开启**定时自动生成**，设置时间与每次条数
2. 确保通知中心已开启**系统公告**且（可选）开启**邮件通知**
3. 后端需持续运行，调度器才会在设定时间触发
4. 可用「立即执行一次」验证整条链路
