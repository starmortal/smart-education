# 版本 v2.3.0 更新日志

**发布日期**: 2026-05-21

## 🎉 新增功能

### 知识库管理系统
- 新增知识库功能模块，支持创建、编辑、删除知识库
- 支持多种文件格式上传：TXT、Markdown、PDF、Word 文档
- 实现文件向量化处理，支持语义搜索
- 支持添加纯文本内容到知识库
- 支持添加网页链接到知识库
- 提供知识库搜索功能，基于向量相似度检索
- 显示文件处理状态（等待中、处理中、已完成、失败）
- 统计知识库文件数量和向量数量

### 用户界面优化
- 在左侧导航栏新增知识库入口
- 设计知识库管理页面，包含列表和详情视图
- 提供文件上传、文本添加、链接添加等多种方式
- 实现知识库搜索对话框，展示搜索结果和相似度评分

## 🔧 技术改进

### 后端架构
- 新增 Knowledge 和 KnowledgeFile 数据模型
- 实现文件解析工具类（FileParser）
- 实现向量化工具类（Vectorizer），集成 OpenAI Embeddings API
- 新增知识库控制器和路由
- 支持异步文件处理，避免阻塞请求
- 实现余弦相似度计算，用于向量检索

### 前端架构
- 新增知识库 API 接口模块
- 创建 KnowledgeBase.vue 页面组件
- 集成文件上传功能，支持拖拽和选择
- 实现实时状态更新和进度显示

## 📝 数据库变更

### 新增集合
- `knowledges`: 知识库基本信息
  - userId: 用户 ID
  - name: 知识库名称
  - description: 描述
  - embeddingModel: 向量化模型
  - fileCount: 文件数量
  - vectorCount: 向量数量

- `knowledgefiles`: 知识库文件
  - knowledgeId: 所属知识库 ID
  - name: 文件名称
  - type: 类型（file/url/text/folder）
  - source: 来源路径或链接
  - content: 文本内容
  - vectors: 向量数组
  - status: 处理状态
  - error: 错误信息

## ⚙️ 配置说明

### 环境变量配置
在 `smart-education-backend/.env` 文件中添加：
```env
# SiliconFlow 嵌入模型配置
EMBEDDING_API_KEY=你的API密钥
EMBEDDING_BASE_URL=https://api.siliconflow.com/v1
EMBEDDING_MODEL=Qwen/Qwen3-Embedding-8B
```

### 向量化模型
- 默认使用 SiliconFlow 提供的 Qwen3-Embedding-8B 模型
- 支持任何 OpenAI 兼容的嵌入模型 API
- 可通过环境变量自定义模型和 API 端点

## 📚 使用说明

### 创建知识库
1. 点击左侧导航栏"知识库"进入知识库页面
2. 点击"新建知识库"按钮
3. 输入知识库名称和描述
4. 点击确定完成创建

### 添加文件
1. 选择一个知识库
2. 点击"添加文件"按钮上传文件
3. 系统自动进行向量化处理
4. 处理完成后可进行搜索

### 搜索知识库
1. 在知识库详情页点击"搜索知识库"
2. 输入搜索关键词
3. 查看搜索结果和相似度评分

## 🔄 API 接口

### 知识库接口
- `GET /api/knowledge/user/:userId` - 获取用户的知识库列表
- `POST /api/knowledge` - 创建知识库
- `PUT /api/knowledge/:id` - 更新知识库
- `DELETE /api/knowledge/:id` - 删除知识库

### 文件接口
- `GET /api/knowledge/:knowledgeId/files` - 获取知识库文件列表
- `POST /api/knowledge/:knowledgeId/files` - 上传文件
- `POST /api/knowledge/:knowledgeId/text` - 添加文本
- `POST /api/knowledge/:knowledgeId/url` - 添加链接
- `DELETE /api/knowledge/files/:fileId` - 删除文件

### 搜索接口
- `POST /api/knowledge/:knowledgeId/search` - 搜索知识库
