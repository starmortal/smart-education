# v3.1.0 更新日志

**发布日期**: 2026-05-23

## 🎉 重大更新

### AI 对话界面全新升级
参考 Cherry Studio 的简洁设计理念，对 AI 对话界面进行了全面优化，提供更流畅的对话体验！

---

## ✨ 新增功能

### 1. 简洁对话流设计
- 移除消息边框和背景框，采用更简洁的对话流式布局
- 用户消息和 AI 消息统一左对齐，视觉更清爽
- 增加消息间距，提升阅读体验
- 优化头像和内容的间距布局

### 2. AI 消息快捷操作
- 每条 AI 回复下方新增快捷操作按钮
- **保存到笔记**：一键将 AI 回复保存为学习笔记
- **保存到知识库**：将 AI 回复保存到指定知识库
- **复制**：快速复制 AI 回复内容到剪贴板
- 按钮采用图标样式，鼠标悬停显示提示
- 按钮默认半透明，悬停时完全显示

### 3. 增强 Markdown 渲染
- 支持完整的 GitHub 风格 Markdown (GFM)
- 支持标题、列表、引用、表格等所有元素
- 代码块语法高亮显示
- 链接、图片自动渲染
- 表格样式优化
- 引用块带左侧蓝色边框

### 4. 快速新建对话
- 输入框左侧新增"新建对话"按钮
- 与上传文件、知识库按钮并列显示
- 采用纯图标样式（Plus 图标）
- 鼠标悬停显示"新建对话"提示
- 点击即可在当前助手下快速创建新对话
- 自动切换到新创建的对话

### 5. 智能内容保存
- 保存到笔记时自动添加"AI对话"标签
- 保存到知识库时自动生成带时间戳的文件名
- 支持选择目标知识库
- 复制功能自动过滤 HTML 标签，保留纯文本

## 📋 API 接口

### 前端新增 API
- `addNote(data)` - 创建笔记

## 🎯 功能说明

### 如何使用快捷操作

1. **保存到笔记**：
   - 点击 AI 回复下方的"保存到笔记"按钮
   - 系统自动创建笔记，标题为"来自 AI 对话"
   - 自动添加"AI对话"标签
   - 保存成功后显示提示消息

2. **保存到知识库**：
   - 点击 AI 回复下方的"保存到知识库"按钮
   - 在弹出的对话框中选择目标知识库
   - 系统自动生成带时间戳的文件名（可修改）
   - 点击"确定"完成保存

3. **复制内容**：
   - 点击 AI 回复下方的"复制"按钮
   - 系统自动提取纯文本内容
   - 复制到剪贴板，可直接粘贴使用

4. **快速新建对话**：
   - 点击输入框左侧的"+"图标按钮
   - 系统在当前助手下创建新对话
   - 自动切换到新对话，可立即开始聊天
