## 🎯 本次更新概述

---

## 🐛 Bug 修复

### 意见反馈历史列表

- **原因**：后端统一返回 `{ code, message, data }`，前端使用裸 `axios` 却读取 `res.data.list`，实际应为 `res.data.data.list`
- **修复**：新增 `api/feedback.js`，改用项目 `request` 封装正确解析 `data.list`
- **后端**：历史接口返回完整内容与截图字段，便于左侧列表与详情展示

### 提交后按钮一直 loading

- **原因**：提交接口在返回前 `await` 发送管理员邮件，SMTP 慢或失败会导致响应迟迟不返回；前端又 `await` 刷新历史后才结束 `loading`
- **修复**：邮件改为异步发送，接口保存后立即返回；前端刷新历史不阻塞提交按钮 loading
