# 版本 v4.3.0 更新日志

**发布日期**: 2026-05-24

---

## 🎯 本次更新概述

更新日志页重构为文档站风格，从 `docs/changelog` 目录动态加载 Markdown，支持版本切换与右侧目录导航。

---

## ✨ 新功能

### 更新日志文档站

- 移除页面内硬编码的旧版更新记录
- 路由仍为 `/changelog`，登录页底部「更新日志」入口逻辑不变
- **左侧**：版本列表（按版本号倒序，显示发布日期与概述摘要）
- **中间**：Markdown 渲染正文（支持表格、列表、标题等）
- **右侧**：当前文档目录（h2/h3 锚点，点击平滑滚动）

### 后端 API

- `GET /api/changelog/list`：读取 `docs/changelog` 下全部版本文件
- `GET /api/changelog/:version`：返回指定版本 Markdown 内容与目录结构
