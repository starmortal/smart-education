# 版本 v3.2.0 更新日志

**发布日期**: 2026-05-23

## 🎨 界面优化

### 知识库页面
- **左侧列表简化**
  - 移除知识库列表中的文件数和向量数显示
  - 界面更加简洁清爽，聚焦知识库名称
  - 系统知识库显示"系统"标签，便于识别

- **系统知识库保护**
  - 系统固定的知识库（如学科知识库）不可删除
  - 系统知识库的删除选项在菜单中隐藏
  - 后端API拦截系统知识库的删除请求
  - 显示"系统"标签标识系统知识库

- **添加文本功能全面升级**
  - 使用专业的 Markdown 编辑器替代原有的简单文本框
  - 对话框尺寸优化：宽度 80%，高度 70vh，提供舒适的编辑空间
  - 对话框垂直居中显示，视觉效果更佳
  - 文件名输入框移至对话框标题栏右侧，布局更合理
  - 自定义关闭按钮，避免重复显示
  - 底部只保留取消和保存按钮，界面更简洁

- **Markdown 编辑器功能**
  - 支持富文本编辑：加粗、斜体、下划线、删除线
  - 支持标题、上标、下标、引用、列表（有序/无序）
  - 支持代码行、代码块、链接、图片、表格
  - 支持撤销/重做操作
  - 支持全屏编辑和预览模式
  - 支持目录导航
  - 实时预览 Markdown 渲染效果

## 📝 技术改进

### 前端优化
- 引入 `md-editor-v3` 组件，提供专业的 Markdown 编辑体验
- 优化对话框布局：
  - 使用自定义 header 插槽实现标题和文件名输入的组合
  - 对话框采用 flex 布局，确保编辑器占满可用空间
  - 添加边框分隔线，视觉层次更清晰
  - 设置 `flex-shrink: 0` 确保头部和底部不被压缩
  - 隐藏默认关闭按钮，使用自定义关闭图标
- 编辑器工具栏配置完整，涵盖常用 Markdown 功能
- 对话框尺寸优化：宽度 80%，高度 70vh，垂直居中显示

## 🔧 修改文件

### 前端
- `frontend/src/views/KnowledgeBase.vue` - 简化列表显示，升级文本编辑器，添加系统知识库保护

### 后端
- `smart-education-backend/controllers/knowledgeController.js` - 添加系统知识库删除拦截
