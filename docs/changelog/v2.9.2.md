# v2.9.2 更新日志

**发布日期**: 2026-05-22

## 🎨 界面重构

### 学习社区页面
- 完全重构页面布局，采用与其他页面一致的左右分栏设计
- 移除顶部统计栏和鼓励讨论提示文字
- 统计数据移至左侧边栏顶部，采用卡片样式展示
- 状态筛选功能移至左侧边栏，采用与错题集相同的卡片交互
- 用户信息和收藏列表整合到左侧边栏
- 添加侧边栏折叠/展开功能，支持隐藏左侧内容
- 顶部标题栏高度统一为 56px，与其他页面保持一致
- 问题列表采用 2列2行 网格布局，保持彩色边框设计

## 📐 布局优化

### 左侧边栏（280px，可折叠）
- **数据统计区域**：疑问总数、未解决数、我的疑问数
- **状态筛选区域**：待解决/已解决快速筛选
- **用户信息卡片**：头像、昵称、学校、年级
- **我的收藏列表**：显示前3条收藏，支持管理

### 右侧主内容区
- **顶部标题栏**：折叠按钮 + 页面标题 + 分页 + 操作按钮
- **问题列表**：网格布局，保持原有交互功能
- **所有对话框**：保持原有功能不变

## 🔧 技术改进

### 布局系统
- 采用固定定位布局（position: fixed）
- 侧边栏宽度 280px，折叠使用 transform 动画
- 主内容区自适应，折叠后自动扩展
- 滚动区域独立，性能更优

### 交互优化
- 折叠动画流畅自然（0.3s transition）
- 卡片悬停效果统一
- 筛选状态实时响应
- 收藏数量从2条增加到3条

### 样式规范
- 顶部标题栏：16px 20px 内边距
- 标题字体：16px / 600 / #303133
- 卡片圆角：8px / 12px
- 边框颜色：#e4e7ed
- 悬停效果：#e8f4ff 背景 + 2px 位移

## 🎯 用户体验提升

### 空间利用
- 折叠侧边栏后，问题列表显示区域更大
- 统计数据和筛选功能集中管理，操作更便捷
- 用户信息和收藏始终可见，无需滚动

### 视觉一致性
- 与错题集、AI对话、知识库页面设计完全统一
- 用户在不同页面间切换时，操作习惯保持一致
- 降低学习成本，提升整体使用体验

### 响应式支持
- 小屏幕下自动调整布局
- 992px 以下自动折叠侧边栏
- 问题列表自适应列数
