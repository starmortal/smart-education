# 版本 v2.5.0 更新日志

## 📅 发布日期
2026-05-21

## 🗑️ 功能移除

### 移除 AI答题页面
- 删除 AIAnswer.vue 页面文件
- 移除 /ai-answer 路由配置
- 从顶部导航栏移除"AI答题"菜单项
- 从侧边导航栏移除"AI答题"菜单项
- 更新首页主按钮，从"立即体验 AI 答疑"改为"立即体验 AI 助手"
- 更新登录页产品链接，从"AI 答疑"改为"AI 助手"

### 移除首页页面
- 删除 HomePage.vue 页面文件
- 移除 /home 路由配置
- 从顶部导航栏移除"首页"菜单项
- 从侧边导航栏移除"首页"菜单项
- 更新 TopNavBar 和 SideNavBar 的 goToHome 函数，跳转到 /ai-chat
- 更新 AdminFeedback 页面的"回到首页"按钮，跳转到 /ai-chat

## 🔧 技术改进

### 前端优化
- 简化导航结构，减少冗余功能入口
- 统一使用 AI助手 作为主要AI交互入口和默认首页
- 优化用户体验，避免功能重复
- 精简页面数量，提升应用性能

## 📝 路由变更

### 移除路由
- DELETE /ai-answer - AI答题页面路由
- DELETE /home - 首页路由

## ⚠️ 注意事项
- 用户访问 /ai-answer 或 /home 路由将无法找到页面
- 根路径 `/` 已重定向到 `/ai-chat` (AI助手)
- 建议使用 /ai-chat (AI助手) 作为主要入口
- 后端 AI 相关接口保持不变，可继续被其他功能使用
- 首页的学习计划展示和实时时间功能已移除
