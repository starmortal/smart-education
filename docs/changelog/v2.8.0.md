# 版本 v2.8.0 更新日志

**发布日期**: 2026-05-22

## 📝 我的笔记功能优化

### 新增功能
- 支持文件夹和笔记的树形嵌套结构
- 左侧列表支持收缩/展开功能
- 笔记右键菜单（重命名、编辑标签、删除）
- 文件夹管理功能
- 自定义右键菜单样式

### 功能优化
- 简化左侧工具栏，删除收藏按钮
- 笔记列表只显示文件名，界面更简洁
- 删除分类功能，保留标签功能
- 优化右侧编辑区顶部布局，只显示文件名和收缩按钮
- 删除操作改为右键菜单触发
- 标签编辑功能移至右键菜单，删除右侧编辑区的标签编辑区域
- 左侧列表选中笔记时显示浅蓝色背景高亮效果

### 界面优化
- 左侧列表采用 Element Plus el-tree 组件展示树形结构
- 工具栏按钮更紧凑（新建笔记、新建文件夹、排序、搜索）
- 右侧顶部只显示文件名和收缩/展开按钮
- 树形列表节点悬停和选中状态优化
- 自定义右键菜单样式，支持重命名、编辑标签、删除操作

### 数据模型调整
- 笔记模型新增 `parentId` 字段（支持文件夹嵌套）
- 笔记模型新增 `isFolder` 字段（区分文件夹和笔记）
- 删除 `noteCategory` 字段
- 保留 `noteTags` 字段

### 后端优化
- 删除笔记/文件夹时支持递归删除所有子项
- 优化笔记列表查询，支持树形结构数据返回
- 删除分类相关接口，保留标签接口

- `frontend/src/views/NotePage.vue` - 完全重构笔记页面
- `frontend/src/api/index.js` - 删除分类 API
- `smart-education-backend/models/Note.js` - 数据模型更新
- `smart-education-backend/routes/note.js` - 后端路由更新
