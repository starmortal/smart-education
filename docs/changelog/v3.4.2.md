## 📝 错题集页面重构

### 布局优化

#### 1. 复习日历位置调整

- **新位置**: 左侧边栏下方
- **优势**: 
  - 更符合用户视觉习惯
  - 左侧筛选区域功能更集中
  - 右侧内容区域更宽敞

#### 2. 日历组件升级
**替换方案**: 自定义 `ReviewCalendar` → Element Plus `el-calendar`

**新增功能**:
- **年月切换控制**
  - 双箭头按钮：切换年份（上一年/下一年）
  - 单箭头按钮：切换月份（上一月/下一月）

- **整月显示**

- **错题数量徽章**

- **多种单元格状态**

#### 3. 日期筛选功能
- **点击交互**: 点击日历中的某一天
- **筛选逻辑**: 右侧只显示该天添加的错题
- **标题更新**: 显示"YYYY-MM-DD 的错题"
- **清除筛选**: 新增"查看全部"按钮
- **分页重置**: 筛选后自动回到第一页

## 🔧 技术实现

### 日历数据加载

#### API接口
```javascript
GET /api/error-book/calendar/:userId?year=2026&month=5
```

**响应格式**:
```json
{
  "code": 200,
  "data": {
    "dates": [
      { "date": "2026-05-01", "count": 3 },
      { "date": "2026-05-15", "count": 5 },
      { "date": "2026-05-23", "count": 2 }
    ]
  }
}
```

#### 降级方案
如果API请求失败，从当前错题列表计算：
```javascript
function calculateErrorCountFromList() {
  const countMap = {};
  errorList.value.forEach(error => {
    const dateStr = dayjs(error.addTime).format('YYYY-MM-DD');
    countMap[dateStr] = (countMap[dateStr] || 0) + 1;
  });
  errorCountByDate.value = countMap;
}
```

### 日期筛选逻辑

```javascript
const displayErrors = computed(() => {
  if (!selectedDate.value) {
    return errorList.value;
  }
  
  return errorList.value.filter(error => {
    const errorDate = dayjs(error.addTime).format('YYYY-MM-DD');
    return errorDate === selectedDate.value;
  });
});
```

### 分页计算

```javascript
const paginatedErrors = computed(() => {
  const start = (currentPage.value - 1) * pageSize.value;
  const end = start + pageSize.value;
  return displayErrors.value.slice(start, end);
});
```
