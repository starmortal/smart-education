# 版本 v4.1.3 更新日志

**发布日期**: 2026-05-23

---

## 🎯 本次更新概述

学习计划页交互对齐学习社区：点击计划左滑进入详情，详情内直接编辑/删除，不再使用弹窗。

---

## ✨ 功能优化

### 左滑查看计划详情

- 点击时间轴计划卡片，主内容区**向左滑动**进入详情页（与学习社区一致）
- 顶部「返回计划列表」返回时间轴
- 当前选中计划在列表中高亮

### 详情内联编辑与删除

- 详情页点击 **「编辑计划」** 切换为同页可编辑表单，无需弹窗
- 支持保存、取消、删除
- 「管理计划」中点击编辑同样进入左滑详情并直接进入编辑模式

### 新增计划

- 仍通过侧边栏「+」按钮打开弹窗（仅用于新增）

---

### 通知中心跳转计划详情

- AI 计划生成通知携带 `planId` / `planIds`
- 点击通知跳转 `/study-plan?planId=xxx`，自动左滑打开对应计划详情页
- 旧通知无 planId 时仍跳转计划列表页

---

## 🔧 技术改动（补充）

- `aiPlanSchedulerService`：通知 `relatedData` 写入计划 ID
- `Notification` 模型：`relatedData` 扩展 `planId`、`planIds`；`relatedType` 增加 `studyPlan`
- `StudyPlan.vue`：根据路由 `planId` 自动打开左滑详情
- `NotificationCenter.vue`：点击 AI 计划通知带 query 跳转
