# 版本 v3.2.2 更新日志

**发布日期**: 2026-05-23

## ✨ 新功能

### 知识库页面
- **实现 URL 内容抓取功能**
  - 使用 axios 发送 HTTP 请求获取网页内容
  - 使用 cheerio 解析 HTML 并提取文本内容
  - 智能提取网页标题和主要内容
  - 自动移除脚本、样式、导航、页脚等无关内容
  - 支持多种常见内容容器选择器
  - 清理多余空白和换行，优化文本格式

- **增强 PDF 和 Word 文档解析**
  - 实现 PDF 文档内容提取（使用 pdf-parse）
  - 实现 Word 文档内容提取（使用 mammoth）
  - 已安装的依赖库现在可以正常工作

## 🐛 Bug 修复

### URL 抓取
- **修复 URL 无法抓取内容的问题**
  - 原因：parseURL 方法未实现，只返回提示信息
  - 解决：完整实现 URL 抓取和内容解析功能
  - 添加详细的错误处理和日志记录

- **优化错误提示**
  - 网址无法访问：提示检查 URL 是否正确
  - 访问超时：提示稍后重试
  - 服务器错误：显示具体的 HTTP 状态码
  - 其他错误：显示详细错误信息

## 📝 技术改进

### 后端优化
- 引入 axios 和 cheerio 库进行网页抓取
- 设置 30 秒超时时间，避免长时间等待
- 添加 User-Agent 头，模拟浏览器访问
- 智能内容提取：
  - 优先从 article、main 等语义化标签提取
  - 支持常见的内容容器 class 和 id
  - 自动清理无关元素（脚本、样式、导航等）
- 文本清理：
  - 移除多余空白字符
  - 规范化换行符
  - 保留文本结构
- 完善日志记录，便于调试和监控

### 内容格式化
- 提取的内容以 Markdown 格式保存
- 标题使用 H1 格式
- 内容与标题之间添加空行分隔
- 便于后续的向量化和检索

## 🔧 修改文件

### 后端
- `smart-education-backend/utils/fileParser.js` - 实现 URL 抓取和文档解析功能
- `smart-education-backend/package.json` - 更新 cheerio 版本
