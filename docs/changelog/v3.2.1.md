# 版本 v3.2.1 更新日志

**发布日期**: 2026-05-23

## 🐛 Bug 修复

### 知识库页面
- **修复中文文件名乱码问题**
  - 在 multer 的 fileFilter 中提前处理文件名编码
  - 使用 Buffer 转换解决 latin1 到 utf8 的编码问题
  - 确保上传文件时中文文件名正确显示和保存
  - 修复电脑导入文件时的中文乱码问题

- **修复添加链接功能**
  - 确认后端 API 路由配置正确
  - 验证添加链接的完整流程
  - 添加错误处理和日志记录

- **优化搜索功能**
  - 搜索功能改为仅搜索文件名
  - 使用正则表达式实现不区分大小写的模糊搜索
  - 移除向量相似度搜索，提升搜索速度
  - 优化搜索结果显示：显示文件类型、状态、创建时间
  - 搜索结果支持悬停高亮效果

- **新增文件状态刷新功能**
  - 在文件列表操作列添加刷新按钮（图标）
  - 点击刷新按钮可实时查看文件处理状态
  - 方便用户查看上传文件的处理进度
  - 删除按钮也改为只显示图标，界面更简洁

## 📝 技术改进

### 后端优化
- 在 multer 的 fileFilter 中提前处理中文文件名编码
- 统一在文件上传入口处理编码问题
- 简化搜索逻辑，使用 MongoDB 正则查询
- 移除对 Vectorizer 的依赖（搜索功能）

### 前端优化
- 更新搜索结果展示组件
- 添加文件类型和状态标签
- 优化搜索结果样式，提升用户体验
- 简化 API 调用参数
- 添加文件状态刷新按钮，方便查看处理进度
