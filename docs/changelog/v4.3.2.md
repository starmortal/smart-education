### 管理后台「演示数据」页（`/admin/demo-data`）

- 查看数据包信息与导入状态
- **导入演示数据**：写入 MongoDB 并记录元数据
- **清除演示数据**：按记录 ID 反向删除，删除后可再次导入
- 展示各模块导入数量统计

### 后端 API

- `GET /api/admin/demo-seed/status`：获取导入状态
- `POST /api/admin/demo-seed/import`：导入演示数据
- `DELETE /api/admin/demo-seed`：清除演示数据

## 💡 使用说明

1. 管理员登录后台 → 左侧菜单「演示数据」
2. 点击「导入演示数据」
3. 使用上表演示账号登录用户端，各页面即可看到示例内容
4. 需要重置时点击「清除演示数据」，再重新导入即可
