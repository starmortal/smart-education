# 使用文档

智慧教育平台功能介绍与使用教程。登录后可通过左侧导航栏（约 70px 宽）快速切换各模块；鼠标悬停图标可查看功能名称。

---

## 快速开始

### 注册账号

1. 打开登录页，点击「注册」进入注册页
2. 填写 **QQ 邮箱**（需为 `@qq.com` 后缀），点击「获取验证码」
3. 查收邮件中的 6 位验证码，设置密码（不少于 6 位）并确认
4. 勾选同意《隐私政策》和《服务条款》后完成注册
5. 注册成功后会收到欢迎通知，使用邮箱和密码登录即可

### 登录与导航

- 登录成功后默认进入 **AI 助手** 页面
- 左侧竖条为全局导航，从上到下依次为：AI 助手、知识库、错题集、学习计划、我的笔记、学习社区、意见反馈、个人中心、通知中心、退出登录
- 底部头像区域可进入个人相关功能
- 登录页底部可查看 **更新日志**、**隐私政策**、**服务条款**、**使用文档**（本文档）

### 建议的首次使用顺序

1. 进入 **个人中心**，完善年级、学校、学习科目（后续计划、错题、AI 分析均会用到）
2. 在 **AI 助手** 中创建或选择助手，发起第一次对话
3. 按需使用错题集、学习计划、知识库等功能

## AI 助手

AI 助手是平台核心功能，基于 DeepSeek AI，支持多助手、多话题、流式回答与知识库检索（RAG）。

### 界面说明

- **左侧面板**：两个标签页
  - **助手**：管理所有 AI 助手（系统助手 + 自定义助手）
  - **对话**：当前助手下的话题（对话）列表
- **右侧区域**：当前话题的消息记录与输入框
- 点击左上角按钮可 **折叠/展开** 左侧面板，获得更大对话空间

### 创建与使用助手

1. 在「助手」标签页点击 **新建助手**
2. 填写助手名称、**系统提示词**（人设，如「你是一位耐心的数学辅导老师」）
3. 可选 **关联知识库**：对话时将自动检索这些库中的内容辅助回答
4. 保存后点击该助手，系统会自动切换到「对话」标签并加载话题列表
5. 系统预置助手不可删除；其提示词与知识库为只读

### 话题（对话）管理

1. 选中助手后，在「对话」标签点击 **新建对话** 创建话题
2. 点击话题名称切换对话；每条话题独立保存历史消息
3. 话题右侧菜单支持：**重命名**、**清空消息**、**删除话题**
4. 首次选中某助手且尚无话题时，系统会自动创建一个

### 发送消息

1. 在底部输入框输入问题
2. **Enter** 发送，**Shift + Enter** 换行
3. 点击 📎 可上传 **图片、PDF、Word、TXT** 等附件（支持多选）
4. AI 以 **流式输出** 实时显示回答，支持 Markdown 格式（公式、列表、代码块等）

### 对话级知识库（临时检索）

除助手默认关联的知识库外，还可为 **当前这一次对话** 额外选择知识库：

1. 在输入框上方点击 **知识库图标**
2. 在弹出层中勾选要参与检索的知识库
3. 发送消息时，系统将 **合并** 助手默认库与临时选择的库进行检索

### 消息快捷操作

鼠标悬停在 AI 回复上，可使用：

- **保存到笔记**：将回答内容一键存入「我的笔记」
- **保存到知识库**：选择目标知识库与文件名后归档
- **复制**：复制整条回复文本

### 使用技巧

- 提问尽量具体，说明科目、年级、已知条件，回答更准确
- 复杂题目可上传题目图片或 PDF
- 需要结合个人资料、错题、笔记回答时，先关联相应知识库
- 不同科目可创建不同人设的助手，避免上下文混淆

## 知识库

知识库用于存放学习资料，并供 AI 助手 **检索增强（RAG）** 使用。支持文件、链接、纯文本三种来源。

### 创建与管理知识库

1. 点击左侧导航 **知识库** 进入
2. 左栏点击 **创建知识库**，填写名称与描述
3. 点击某个知识库可在右侧查看其文件列表
4. 知识库菜单支持 **编辑名称/描述**、**删除**（删除后不可恢复）

### 添加内容

选中知识库后，顶部工具栏提供：

| 按钮 | 说明 |
|------|------|
| 添加文件 | 上传 `.txt`、`.md`、`.pdf`、`.doc`、`.docx`，单文件不超过 10MB |
| 添加链接 | 填写名称与网页 URL，系统自动抓取并处理 |
| 添加文本 | 使用 Markdown 编辑器直接编写并保存 |
| 搜索 | 在当前知识库内语义搜索，查看匹配片段 |

上传或添加后，文件会经历 **处理中 → 已完成** 状态；仅处理完成的文件可被 AI 检索。

### 与 AI 助手联动

1. 编辑助手时，在「关联知识库」中勾选默认使用的库
2. 或在对话时通过 **临时知识库** 按钮追加
3. AI 回答时会优先检索相关内容，并在合适时引用资料

### 使用建议

- 按科目或专题分库，如「高一数学公式」「英语作文素材」
- 定期整理失效链接与重复文件
- 从 AI 对话中 **保存到知识库** 可快速沉淀优质解答

## 错题集

错题集帮助系统记录、分类和复习做错的题目，并支持 AI 分析薄弱点。

### 界面说明

- **左侧**：科目筛选、掌握状态筛选（未掌握 / 已掌握等）、添加错题按钮
- **中间**：错题卡片列表，顶部显示 **全局统计**（总数、未掌握、已掌握，不受当前筛选影响）
- **右侧/详情**：点击卡片查看题目详情、解析、错误原因等

### 添加错题

1. 点击左侧 **添加错题**（+ 按钮）
2. 填写 **题目内容**、**所属科目**、**题型**（选择/填空/解答等）
3. 填写 **错误原因**、**正确答案**、**解析**（可选但建议填写）
4. 确认添加后，错题默认状态为 **未掌握**

### 复习与筛选

1. 左侧点击 **未掌握** / **已掌握** 快速筛选
2. 点击科目名称按科目查看
3. 点击顶部 **筛选** 可按题型、掌握程度、添加时间等组合条件
4. 打开错题详情后，复习完毕点击 **标记掌握** 更新状态

### AI 错题分析

1. 点击顶部 **AI 分析** 按钮
2. 系统根据你的错题数据生成 **分析报告**（薄弱科目、常见错误类型、复习建议等）
3. 报告支持 Markdown 渲染，可结合筛选结果针对性查看

### 使用建议

- 错题录入越完整，AI 分析越准确
- 定期将已掌握题目标记，聚焦未掌握部分
- 可与学习计划配合，在计划中安排错题复习任务

## 学习计划

学习计划用于管理学习目标与进度，支持 **时间线视图** 与 **AI 智能制定**。

### 界面说明

- **左侧**：按状态（未开始 / 进行中 / 已完成 / 已逾期）和 **科目** 筛选；科目列表来自个人中心设置
- **右侧**：时间线展示计划卡片，按日期分组；点击卡片可查看详情并编辑

### 手动创建计划

1. 点击左侧 **新增计划**（+ 按钮）或顶部相关入口
2. 填写 **计划标题**、**科目**、**开始/截止时间**、**进度**（0–100%）
3. 保存后出现在时间线对应日期分组中
4. 拖动或编辑进度条时，系统自动识别状态：
   - 0% → 未开始
   - 1%–99% → 进行中
   - 100% → 已完成
   - 已过截止日且未完成 → 已逾期

### 查看与管理

1. 时间线按 **今天 / 未来 / 过去** 等分组展示，可折叠展开
2. 点击计划卡片进入详情，可修改标题、时间、进度、备注
3. 左侧 **管理计划** 支持批量操作
4. 卡片上显示剩余天数、进度条；AI 制定的计划会展示 **AI 制定理由**

### AI 智能制定

点击顶部 **AI 智能制定** 打开抽屉面板：

1. **学习画像**：系统综合个人资料、考试成绩、错题、历史计划等生成摘要；可点击「刷新画像」
2. **生成计划草案**：设置计划数量、时间范围、重点科目等，AI 生成多条草案
3. 预览草案后 **确认导入**，计划写入时间线
4. **定时自动生成**（可选）：
   - 开启后每天在设定时间自动生成计划
   - 可设置每次生成条数
   - 支持 **立即执行一次** 测试
   - 生成完成后在 **通知中心** 提醒，并可选邮件通知

### 使用建议

- 请先在个人中心设置 **学习科目**，否则科目筛选为空
- 大目标拆成多条短计划，便于跟踪进度
- 结合 AI 制定理由理解计划安排逻辑，必要时手动微调

## 我的笔记

笔记模块提供 **文件夹 + 笔记** 的树形结构，支持 Markdown 编辑、标签与搜索。

### 基本操作

1. 进入 **我的笔记** 页面
2. 顶部工具栏：
   - **新建笔记**：创建一篇 Markdown 笔记
   - **新建文件夹**：创建目录，可嵌套组织内容
   - **排序**：按更新时间等排序
   - **搜索**：按标题/内容搜索
3. 左侧树形列表点击笔记，右侧打开 **Markdown 编辑器**
4. 内容修改后约 **1 秒自动保存**，无需手动点击保存

### 文件夹与标签

1. **右键** 笔记或文件夹可：重命名、编辑标签、删除
2. 标签支持多选与自定义新建，顶部可按标签 **筛选** 笔记
3. 文件夹仅用于分类，不能直接编辑正文

### 编辑器功能

- 支持标题、加粗、列表、代码块、表格、链接、图片等 Markdown 语法
- 支持预览、全屏编辑、目录导航
- 可从 AI 助手 **保存到笔记** 快速沉淀对话内容

### 使用建议

- 用文件夹区分「课堂笔记」「复习提纲」「AI 摘录」等
- 给笔记打标签（如 `#数学` `#期中`）便于后期检索
- 重要笔记可定期导出或同步到知识库供 AI 检索

## 学习社区

学习社区是同学互助问答平台，适合 AI 难以解决的开放性讨论题。

### 浏览与提问

1. 进入 **学习社区**，默认展示问题列表
2. 可按 **待解决 / 已解决** 等状态筛选
3. 点击 **发起提问**，填写标题、详细描述、标签后发布
4. 点击问题卡片进入详情页

### 回答与互动

1. 在问题详情底部输入 **回答** 并提交
2. 可对他人回答进行 **点赞**
3. 问题发布者可将有帮助的回答设为 **最佳答案**
4. 支持对回答进行 **嵌套回复**（楼中楼）

### 收藏管理

1. 在问题详情或列表中 **收藏** 有价值的问题
2. 切换到 **我的收藏** 标签统一管理
3. 支持单条或 **批量取消收藏**

### 使用建议

- 提问时写清背景、已尝试的方法，更容易获得有效回答
- 回答请文明、准确，引用公式或步骤时分段说明
- 收藏优质问答，可作为后续复习资料

## 个人中心

个人中心管理账号信息与学习数据，是 AI 分析、计划制定的重要数据来源。

### 完善个人资料

1. 点击 **编辑资料**
2. 可修改：昵称、头像、学校、**年级**、**学习科目**（多选）
3. 学习科目会影响：学习计划科目筛选、AI 画像、部分统计展示
4. 保存后全平台同步

### 考试成绩管理

1. 右侧 **考试成绩** 区域点击 **添加考试**
2. 填写考试名称、日期、各科分数
3. 可设置各科 **满分** 便于计算得分率
4. 支持查看、编辑、删除历史考试记录
5. 成绩数据用于 **趋势图表** 与 AI 分析

### AI 成绩分析

1. 点击 **AI 成绩分析** 按钮
2. 选择要分析的考试（或全部考试平均）
3. AI 生成评价报告：整体水平、优势科目、薄弱科目、学习建议等
4. 历史分析记录在页面底部 **AI 成绩分析历史** 中可回顾

### 使用建议

- 注册后尽快完善年级与科目，解锁完整功能
- 每次大考后录入成绩，便于观察长期趋势
- 定期使用 AI 分析，对照调整学习计划

## 通知中心

通知中心集中展示平台消息，支持实时推送与个性化设置。

### 查看通知

1. 点击左侧导航 **通知中心**（或侧边栏铃铛图标）
2. 分 **全部 / 未读 / 已读** 三个标签浏览
3. 点击单条通知可查看详情；部分通知可 **跳转到相关页面**（如社区回复、系统公告）
4. 支持 **标记已读**、**全部已读**、删除单条、**清空通知**

### 通知类型

| 类型 | 说明 |
|------|------|
| 注册欢迎 | 新用户注册成功 |
| 登录提醒 | 账号登录时 |
| 社区回复 | 你的问题被回答或评论 |
| 点赞 | 你的回答被点赞 |
| 关注 | 被其他用户关注 |
| 系统公告 | 平台重要通知 |
| AI 计划生成 | AI 自动/手动生成学习计划完成 |

### 通知设置

1. 点击 **通知设置** 打开面板
2. **邮箱通知** 总开关：控制是否接收邮件提醒
3. 可单独开关各类型通知的邮件推送
4. 设置实时保存

### 使用建议

- 开启 AI 计划定时生成时，建议同时开启「AI 计划生成」通知
- 定期清理已读通知，保持列表整洁
- 系统公告请点击查看完整内容

## 常见问题

### 登录相关

- **忘记密码**：在登录页使用找回密码流程（通过注册邮箱验证）
- **验证码收不到**：检查 QQ 邮箱垃圾箱，确认邮箱格式正确，60 秒后可重新获取

### AI 助手相关

- **回答不准确**：补充题目条件、关联知识库，或换用更专业的助手提示词
- **知识库未生效**：确认文件状态为「已完成」，且助手或对话已关联该库
- **流式输出中断**：检查网络后重试发送

### 数据与隐私

- 学习数据（计划、错题、笔记等）仅与你的账号关联
- 删除账户后数据按《隐私政策》处理
- 详细说明见 **平台文档 → 隐私政策**

### 更多帮助

- **更新日志**：了解各版本新功能与修复（登录页底部入口）
- **服务条款**：使用规范与免责声明
